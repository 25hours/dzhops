# dzhops   
+ 使用Django框架开发的Salt Stack Web UI   

##环境：   
+ salt-master 2014.7.0     
+ salt-minion 2014.7.0      
+ salt-api 2014.7.0     
+ Django 1.6.8     
+ python 2.6     
+ MySQL 5.5
